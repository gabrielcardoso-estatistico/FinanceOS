## 📊 Dashboard Financeiro — FinanceOS

Dashboard interativo para análise financeira, com foco em **KPIs, visualização de dados e tomada de decisão**.

---

## 🚀 O que o projeto mostra

* 💰 Receita, Despesa, Lucro e Margem
* 📈 Comparativo mensal (Receita vs Despesa)
* 📊 Distribuição de gastos por categoria
* 📉 Evolução do saldo (30 dias)
* 🧾 Transações recentes

---

## ⚙️ Tecnologias

* Python (geração de dados)
* HTML + CSS
* JavaScript
* Chart.js

---

## ▶️ Como executar

```bash
python script.py
```

Abra o arquivo gerado:

```
dashboard_financeiro.html
```

Demonstrar habilidades em:
